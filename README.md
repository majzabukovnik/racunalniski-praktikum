# Računalniški praktikum
#### 1. **Multiple cursors / več kazalcev**
- **macOS:** `Option + Click`
- **Opis:** Omogoča urejanje več vrstic hkrati (npr. dodajanje istega besedila na več mestih).  

---

#### 2. **Command Palette**
- **macOS:** `Cmd + Shift + P`
- **Opis:** Odpre centralno iskalno okno, kjer lahko dostopaš do vseh ukazov v VS Code, vključno z nastavitvami, razširitvami in ukazi brez uporabe miške.  

---

#### 3. **Zlaganje / razpiranje kode (Code Folding)**
- **macOS:** `Cmd + K Cmd + 0` / `Cmd + K Cmd + J`
- **Opis:** Omogoča skrivanje ali razpiranje blokov kode, kar je uporabno pri pregledovanju večjih datotek.
